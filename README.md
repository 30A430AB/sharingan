## 快速开始

### 环境要求
- Python 3.10.6 [下载](https://www.python.org/downloads/release/python-3106/)

### 下载
```bash
git clone https://github.com/30A430AB/MangaTransFer.git&&cd MangaTransFer
```

### 运行
```bash
python launcher.py
```
初次运行需点击配置页安装依赖和下载资源
![launcher](docs/images/launcher.webp)

#### 工具说明
- 修复画笔：按下鼠标左键拖动抹除文字
- 还原画笔：按下鼠标左键拖动清除修复结果
- 矩形工具：按下鼠标左键拖动矩形框抹除框内文字
